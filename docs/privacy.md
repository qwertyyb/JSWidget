# 隐私政策

最后更新日期：2026 年 5 月 22 日

JSWidget（以下简称"本应用"）尊重并保护用户的隐私。本隐私政策说明本应用如何收集、使用和保护您的信息。

## 信息收集

本应用可能在您授权后访问以下信息：

### 位置信息

当您在脚本中使用位置相关 API 时，本应用会请求您的位置权限。位置数据仅在本地用于脚本执行和 Widget 显示，不会上传到任何服务器。

### 健康数据

当您在脚本中使用健康相关 API 时，本应用会请求访问 HealthKit 数据的权限。健康数据仅在本地用于 Widget 显示，不会上传到任何服务器。

### 照片

当您为脚本选择图片资源时，本应用会请求访问相册的权限。所选图片仅存储在本地脚本包中。

## 数据存储

- 所有用户脚本和相关数据存储在设备本地或您的 iCloud 私有容器中
- 本应用不使用任何第三方分析或广告 SDK
- 本应用不会将您的个人数据传输到任何外部服务器

## 网络请求

本应用中的网络请求仅来自用户编写的脚本（通过 `$http` / `$fetch` API）。本应用本身不会主动发起网络请求收集用户数据。

## iCloud 同步

如果您启用了 iCloud，脚本文件会通过 Apple iCloud 服务在您的设备间同步。这完全由 Apple 的基础设施处理，本应用不会访问或存储您的 iCloud 凭证。

## 用户追踪

本应用不追踪用户，不收集设备标识符，不使用任何广告追踪技术。

## 第三方服务

本应用不集成任何第三方分析、广告或社交 SDK。

## 儿童隐私

本应用不针对 13 岁以下儿童，也不会有意收集儿童的个人信息。

## 隐私政策变更

如本隐私政策有更新，我们会在此页面发布修订版本。建议您定期查看本页面。
